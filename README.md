# LLM 100 核心概念 · 费曼学习法

> 大语言模型 100 个核心概念，按重要性排列。每个概念用费曼学习法掌握——自己定义，而不是背定义。

## 项目结构

```
llm-concepts-100/
├── README.md                    ← 你在这里
├── 100-concepts-full.md         ← 100 个概念的完整词条
├── refined-definitions.md       ← 你自己的精炼定义（持续更新）
├── progress.yaml                ← 学习进度
├── concepts/                    ← 每个概念一个文件
│   ├── 01-token.md
│   ├── 02-transformer.md
│   └── ... (100 个)
└── daily/                       ← 每日学习日志
    ├── day-01.md
    └── ...
```

## 学习方法

每个概念三步：

1. **我讲** → Hermes 用大白话解释
2. **你讲** → 用你自己的话、带比喻、不准用术语，解释回来
3. **修正** → Hermes 挑漏洞 → 你重新定义 → 直到精准

**标准：** 能用一句话让完全不懂的人听懂，才算真正掌握。

| 第五梯队 (81-100) | 多模态 → MoE → Skill → MCP → ... | LLM 生态与应用 |
